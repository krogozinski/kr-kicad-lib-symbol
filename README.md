# KiCad symbol library
Each component symbol has a separate library file to simplify tracking of changes.

## List of components
```bash
├── Battery_Management
│   └── BQ21040
├── Connector
│   └── PMOD
├── Regulator_Linear
│   └── TPS7A05
├── Regulator_Switching
│   └── TPS63031
├── Sensor_Motion
│   ├── ADXL345
│   ├── BMG250
│   ├── LIS3DH
│   ├── LSM6DS33
│   └── MMA8451Q
```
